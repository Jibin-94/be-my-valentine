## Setup Instructions

### 1. Add Your Photos
Replace these placeholder images with your own photos:
- `photo1.jpg`
- `photo2.jpg`
- `photo3.jpg`

Place them in the same folder as index.html

### 2. Add Background Music
Download a free romantic music MP3 and save it as `romantic-music.mp3` in the same folder.

**Free Music Sources:**
- YouTube Audio Library (https://www.youtube.com/audiolibrary)
- Free Music Archive (https://freemusicarchive.org)
- Pixabay Music (https://pixabay.com/music/)

Search for: "romantic piano", "love song instrumental", "romantic background music"

### 3. Host on GitHub Pages (FREE)

1. Create a GitHub account at https://github.com
2. Create a new repository named `valentine` (or any name)
3. Upload all files: index.html, style.css, script.js, photos, and music
4. Go to repository Settings â†’ Pages
5. Under "Source", select "main" branch
6. Click Save
7. Your site will be live at: `https://yourusername.github.io/valentine`

### Alternative Free Hosting Options:
- **Netlify**: Drag and drop your folder at https://app.netlify.com/drop
- **Vercel**: Import from GitHub at https://vercel.com
- **GitHub Pages**: Follow steps above
